1) Докеризовать минимальное приложение на python (не забыть запушить в docker hub)
```python
from flask import Flask

app=Flask(__name__)

@app.route('/')
def hello():
  return 'Hello from Pod!'

if __name__ == "__main__":
  app.run(host='0.0.0.0', port=8080)
```
Написать манифест, в котором будет описан pod и service для этого приложения. Прокинуть service на хост с помощью 
```bash
kubectl port-forward svc/<svc-name> 8080:8080
```
и проверить что оно работает с помощью curl
2) Докеризовать приложение на golang (не забыть запушить в docker hub)
```go
package main

import (
    "log"
    "net/http"
)

func main() {
    http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        w.Write([]byte("Go API ready!"))
    })
    
    http.HandleFunc("/healthz", func(w http.ResponseWriter, r *http.Request) {
        w.WriteHeader(http.StatusOK)
        w.Write([]byte("OK"))
    })

    log.Println("Server starting on :8080")
    log.Fatal(http.ListenAndServe(":8080", nil)) 
}

```
- написать манифест deployment для этого приложения (количество реплик задать 3) и манифест service для балансировки трафика на поды приложения.
- прокинуть service на хост как в предыдущем задании и проверить crul`ом на двух эндпоинтах
  - `/`
  - `/healthz`
- отмасштабировать deployment в бОльшую и в мЕньшую стороны
- изменить код приложения на такой
```go
package main

import (
    "encoding/json"
    "log"
    "math/rand"
    "net/http"
    "time"
)

func main() {
    rand.Seed(time.Now().UnixNano())

    http.HandleFunc("/", func(w http.ResponseWriter, r *http.Request) {
        w.Write([]byte("Go API ready!"))
    })
    
    http.HandleFunc("/healthz", func(w http.ResponseWriter, r *http.Request) {
        w.WriteHeader(http.StatusOK)
        w.Write([]byte("OK"))
    })

    http.HandleFunc("/data", func(w http.ResponseWriter, r *http.Request) {
        w.Header().Set("Content-Type", "application/json")
        data := map[string]interface{}{"requests": rand.Intn(1000)} 
        json.NewEncoder(w).Encode(data)
    })

    log.Println("Server starting on :8080")
    log.Fatal(http.ListenAndServe(":8080", nil))
}
```
- собрать новую версию image (например, если первая была `goapp`, то эта будет `goapp:v2`; не забыть запушить в docker hub)
- задеплоить новую версию приложения с помощью существующего deployment; проверить работу curl на трех эндпоинтах
  - `/`
  - `/data`
  - `/healthz`
- откатить обратно (через rollout)
