> [!NOTE]  
> приложению требуются переменные, это надо учесть при написании манифестов

1) Докеризовать приложение https://github.com/AnastasiyaGapochkina01/k8s-nodeapp
2) Запустить приложение через docker compose, проверить
3) Написать
- deployment и service для приложения; в deployment обязательно добавить probes; количество реплик - 3
- deployment и service для redis; количество реплик - 1
4) Применить манифесты, проверить что приложение работает
5) Изучить стратегию деплоя "blue-green deployment"
6) Реализовать "blue-green" деплой для приложения с помощью двух deployment; переключение можно осущетсвлять через patch service
```bash
kubectl patch service myapp-service -p '{"spec":{"selector":{"app":"myapp","version":"green"}}}'
```
